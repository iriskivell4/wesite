Every web page includes the following parts: a doctype tag an html element a head element a title element a body element Doctypes All HTML documents require a doctype to tell the browser which version of HTML we are using. It must be the first line in the file. Without it, browsers will still show the document but may have unexpected results. The html Element The html tag is always the first tag in the document. It is the root or parent element. All other elements are stored or “nested” inside. The lang attribute classifies the language that the content will be in. Many lang values are available like fr for French or es for Spanish. The head Element The head element includes information about the page, but does not display content. Things like CSS, JavaScript, and metadata can be included here. The is the only required tag inside of this element. The title Element <!DOCTYPE html> <html lang="en"> <head> <title> The title element lives inside the element. It sets the title of the page which is displayed at the top of the browser tab.
